# Canva Production Scripts - Weeks 01-03

Last updated: 2026-02-11  
Status: Production Draft (ready for Canva build and MP4 export)

## 1) How to use this file
- Build each clip in Canva as a separate 16:9 video project.
- Export each clip as MP4.
- Upload exported files to the teacher dashboard:
  - Clip A file -> `clipA`
  - Clip B file -> `clipB`
- Run class in fixed order: `clip_a -> activity -> clip_b`

## 2) Global production settings (all clips)
- Canvas size: 1920x1080 (16:9)
- Frame rate: 30 fps
- Subtitle style: high contrast, short lines, max 8 words per line
- Dialogue pace: clear, slow, classroom-friendly
- Sound design: low ambient hum, no loud jump effects
- Visual tone: sterile, cheerful, procedural

## 3) File naming standard
- `W01_ClipA_Onboarding_Broadcast_v1.mp4`
- `W01_ClipB_Assignment_Confirmation_v1.mp4`
- `W02_ClipA_Dispatch_Conflict_v1.mp4`
- `W02_ClipB_Official_Reframe_v1.mp4`
- `W03_ClipA_Queue_Pressure_v1.mp4`
- `W03_ClipB_Pressure_Choice_v1.mp4`

## 4) Teacher upload checklist (per week)
1. Export Clip A and Clip B from Canva as MP4.
2. In Teacher Dashboard, open the target week.
3. Upload Clip A file to slot `clipA`.
4. Upload Clip B file to slot `clipB`.
5. Confirm `nowShowingStage` flow in class:
   - start at `clip_a`
   - switch to `activity`
   - switch to `clip_b`

## 5) Download/export checklist (Canva)
1. Share -> Download.
2. File type: `MP4 Video`.
3. Quality: 1080p.
4. Confirm audio included.
5. Use exact naming from Section 3.
6. Save to your media folder and upload to dashboard.
