# Story Arc Timeline (18 Shifts)

Last updated: 2026-03-08

## Purpose
This is the fixed sequencing map for how each class period uses video and atmosphere.
It keeps script and story on one timeline.

## Fixed Playback Rule (Teacher-Guided)
For every shift:
1. Clip A plays first (`nowShowingStage = clip_a`)
2. Students complete the core activity (`nowShowingStage = activity`)
3. Clip B plays as the reveal/hook (`nowShowingStage = clip_b`)

No free-roam sequence in class delivery.

## Media Upload Rule
- Keep video upload open for all weeks.
- Teacher uploads Clip A and Clip B before class (Canva export or direct file).
- Embed URL can remain as fallback, but upload is the primary path.
