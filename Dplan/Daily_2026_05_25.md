## Part 2 — Inscription Pool / Word Pool 2.0

### Path through the day

1. **Freeze fix #1** (commit `72a3780`) — `useGhostTicker` was running
   a `requestAnimationFrame` loop at 60fps, cascading re-renders into
   the parent and choking the text input. Throttled to setInterval
   100ms (10fps), which is plenty for ghost-typing indicators.
   Also fixed `abandoned: false && abandoned` bug that always evaluated

### Locked design decisions

- **Word Pool is a typing RACE, not a quiz.** Per-character coloring
  with auto-advance on exact match. No Enter / Submit button.
- **Racing-track standings at the TOP.** Each citizen has a fixed-width
  24-cell lane (▸ for completed, ▶ for current, · for remaining).
  Self is bright + glow, leader gets ▲ indicator, others dim.
- **Overflow handling: monkeytype-style.** Student can type past target
  length; overflow chars render in red. Auto-submit ONLY on exact
  match. Clear "backspace to correct" hint when input is wrong.
- **Phosphor green palette** (`#33CC66` standard, `#66FF99` bright,
  `#1F8540` dim, `#0F4F26` faint). Classic DOS terminal look.
- **Hybrid drills**: ~60% word warm-ups + ~40% in-world sentences.
  Sentences preferentially use words from the warm-up rounds for
  immediate context reinforcement.
- **Backend identifiers stay `inscription`** — only display says
  "Word Pool". Route remains `/api/inscription/*`, tables remain
  `inscription_drills`, etc.

## How to test the new Word Pool quickly

1. Log in as student `CA-1` / PIN `1234`.
2. Make sure the student's class is unlocked to W4 (teacher dashboard
   → Manage Classes → Weeks).
3. Open Terminal → tap the `[ ].edited` tile if you want to verify
   the W4 hidden app shell. Tap `Word Pool` for the typing race.
4. In Word Pool: hit "Open Pool" or "Solo Practice." First few prompts
   are individual TOEIC words; later ones are full sentences.
5. Type the word — characters turn green as you match. Type a wrong
   character → it shows red with a "> backspace to correct." hint.
   Auto-advance on exact match. No Enter needed.
6. Watch the racing-track at the top — ghost citizens advance as you
   play. You'll see who's ahead and by how many words.
